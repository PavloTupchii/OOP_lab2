## Варіант завдання

Розрахунок параметрів виконується на основі номера залікової книжки (№ = 5120):

1. **C3 (Тип текстових змінних):** 5120 % 3 => остача 2
   * Тип: `StringBuffer`.
2. **C17 (Дія з текстом):** 5120 % 17 => остача 3
   * Завдання: В усіх питальних реченнях заданого тексту знайти та надрукувати без повторень слова заданої довжини.

## Опис реалізації

Програма виконує наступні кроки для обробки тексту:

1. **Обробка виключних ситуацій:** Використання блоку `try-catch` для перевірки вхідних даних на `null` та запобігання критичним помилкам.
2. **Робота з буфером:** Текст ініціалізується як об'єкт класу `StringBuffer` згідно з вимогами варіанту.
3. **Поділ на речення:** Текст розбивається на масив речень за допомогою методу `.split()` та регулярного виразу, що враховує знаки завершення речень (`.`, `!`, `?`).
4. **Фільтрація речень:** Програма перевіряє останній символ кожного речення. Обробці підлягають лише ті, що закінчуються на знак питання.
5. **Робота зі словами:**
   - Кожне питальне речення очищається від пунктуації та розбивається на слова.
   - Відбираються лише слова, довжина яких відповідає заданому параметру.
6. **Видалення повторів:** Знайдені слова зводяться до нижнього регістру та перевіряються на унікальність перед додаванням до фінального списку.
7. **Вивід результату:** Формований список унікальних слів виводиться у консоль.
